プロフィール

コモ トモキ｜イラストレーター

〇〇県在住。広告制作会社でデザイナーとして勤務後、フリーランスのイラストレーターとして活動。 「心がほかほかすること」「小さな驚きがあること」を大切に、本の挿絵や装画を中心に制作しています。

読者の多くが40〜50代となる一般書・実用書・エッセイなど、 落ち着いたトーンの中に、ふっと目を留めたくなるような一枚をご提案します。

得意な分野

  • エッセイ・コラムの挿絵
  • 一般書・実用書の装画・カット
  • Web記事のイラスト・ヘッダー

出版社・編集の方へ

本や紙媒体のお仕事を、特に積極的にお受けしています。 下記の条件・情報が分かると、スムーズにご提案が可能です。

対応可能なジャンル

  • エッセイ・実用書・新書
  • 一般向け文芸書
  • フリーペーパー・会報誌・小冊子

ご相談時にいただきたい情報

  • 書名（仮題でも可）、著者名
  • 想定読者層・年代
  • 判型・ページ数・色数
  • スケジュール（ラフ・決定稿・入稿日）
  • ご予算の目安

納品形式

  • データ納品：PSD / PNG / TIFF / その他ご相談
  • 解像度：300dpi以上（印刷に合わせて調整）
  • カラーモード：CMYK / RGB 応相談

ご依頼の流れ

初めての方にも分かりやすいよう、基本的な進行フローをまとめました。

  1. 1. お問い合わせ

    「お問い合わせフォーム」より、案件内容・スケジュール・ご予算の目安などをお知らせください。 2〜3営業日以内にご返信いたします。

  2. 2. ヒアリング・お見積もり

    メールやオンラインミーティングにて、イメージや点数などを確認のうえ、お見積もり・スケジュールをご提案します。

  3. 3. ラフ案の制作

    ご要望に沿ってラフを制作し、ご確認いただきます。修正回数の目安もこの段階で共有いたします。

  4. 4. 本制作

    ラフ決定後、本制作に入ります。随時確認を行いながら、印刷・入稿に耐えうるデータとして仕上げます。

  5. 5. 納品・請求

    データ納品後、請求書をお送りいたします。二次利用などの条件については事前にご相談ください。

これまでのお取引・お声

許可をいただいた範囲で、順次掲載予定です。
